Under the Hood

How It Works

01 — DETECT

YOLOv8 Object Detection

Every frame is scanned using YOLOv8n to detect cars, trucks, buses, and motorcycles with high accuracy.

02 — TRACK

Multi-Object Tracking

Each vehicle is assigned a persistent ID across frames, letting us follow its path through the video.

03 — MEASURE

Speed Calculation

Pixel displacement per frame is converted to km/h using frame rate and a real-world scale factor.
